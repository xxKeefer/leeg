## Description

Track Bo3 match series between team pairings. Each game within a series has its own result and optional replay. Support manual result entry as fallback.

## Acceptance Criteria

- [ ] Each match is a Bo3 series between two team pairings
- [ ] Individual games within a series are tracked with game number and winner
- [ ] Each game can link to a Showdown replay URL
- [ ] Admin can manually enter game results when replay is unavailable
- [ ] Series winner is determined automatically from game results
